# OSINT Investigation & Threat Analysis (Sanitized Case Study)
## Overview
This project demonstrates the application of OSINT techniques to analyze suspicious online activities and identify potential fraud patterns.
## Objective
To investigate suspicious digital entities and uncover behavioral patterns using publicly available information.
## Tools Used
- Shodan
- WHOIS Lookup
- Wayback Machine
- Google Dorks
## Methodology
- Passive reconnaissance
- Social media analysis
- Cross-platform correlation
- Pattern identification
## Key Findings
- Identified suspicious behavior patterns across multiple platforms
- Detected coordinated activity indicating possible scam networks
- Observed use of duplicate identities and misleading promotions
## Ethical Note
All analysis was conducted using publicly available data. No sensitive or personal information is disclosed.
## Outcome
This project demonstrates practical OSINT skills in reconnaissance and threat analysis.
